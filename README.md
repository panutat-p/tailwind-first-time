# tailwind-first-time

https://tailwindcss.com/docs/installation


```shell
npm i prettier eslint --save-dev
echo tabWidth: 2 > .prettierrc.yaml
npm init @eslint/config

npm i -D tailwindcss
npx tailwindcss init
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```
